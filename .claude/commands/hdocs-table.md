# 한컴독스 표 생성/편집 (/hdocs-table)

한컴독스 웹 에디터에서 표를 생성하고 편집하는 skill.
셀 조작, 테두리/배경, 블록 계산식 등 표 관련 모든 작업을 처리한다.

사용자 요청: $ARGUMENTS

---

## 실행 전 절차

1. **에디터 탭 확인**: `tabs_context_mcp` → 한컴독스 에디터 탭 식별
2. **기존 다이얼로그 닫기**: Escape

---

## A. 표 만들기

### 방법 1 - 메뉴 (권장)
1. 메뉴바에서 "입력" 또는 "표" 메뉴 열기:
   ```
   find(query="입력 메뉴", tabId={탭ID}) → left_click
   ```
   또는
   ```
   find(query="표 메뉴", tabId={탭ID}) → left_click
   ```
2. "표 만들기" 선택:
   ```
   find(query="표 만들기", tabId={탭ID}) → left_click
   ```
3. 표 만들기 다이얼로그에서:
   - 줄 수(행): 입력 필드 찾기 → 값 입력
   - 칸 수(열): 입력 필드 찾기 → 값 입력
   ```
   find(query="줄 수", tabId={탭ID}) → triple_click → type("{행수}")
   find(query="칸 수", tabId={탭ID}) → triple_click → type("{열수}")
   ```
4. 확인 클릭:
   ```
   find(query="확인", tabId={탭ID}) → left_click
   ```

### 방법 2 - 단축키
```
computer(action="key", text="alt+n", tabId={탭ID})
→ wait(0.3)
→ computer(action="key", text="t", tabId={탭ID})
```
→ 표 만들기 다이얼로그 → 행/열 수 입력 → 확인

### "글자처럼 취급" 규칙 ★ 중요
```
□ 표: 기본 체크 → 여러 쪽에 걸치는 표만 해제
□ 그림: 무조건 체크
```
- **체크 시**: 표/그림이 텍스트 흐름에 포함 (줄 바꿈 자동)
- **미체크 시**: 텍스트와 겹칠 수 있음 → Enter로 상하 여백 확보 필수
- 다이얼로그 하단 "기타" 섹션에 체크박스 있음

## B. 셀 이동 / 표 탈출

### 기본 이동
```
다음 셀:     Tab
이전 셀:     Shift+Tab
```

### 방향키 이동
셀 내 텍스트 끝에서 방향키를 누르면 인접 셀로 이동:
```
위: ArrowUp    아래: ArrowDown
왼쪽: ArrowLeft (텍스트 처음에서)    오른쪽: ArrowRight (텍스트 끝에서)
```

### 표 탈출 ★ 검증 완료
```
방법 1 (권장): Shift+Esc → 표 안 어디서든 즉시 표 밖으로 이동
방법 2: 마지막 행 셀에서 ArrowDown
방법 3: 표 아래 본문 영역 직접 클릭
```
- 확인: 스타일 드롭다운이 "바탕글" 등으로 변경되면 표 탈출 성공
- **⚠ Shift+Esc는 표 위쪽으로 탈출** → 표 아래에 텍스트 입력 시 Ctrl+End로 문서 끝 이동 필요

### 특정 셀로 이동
```
방법 1 (권장): screenshot → 셀 텍스트 중앙 좌표 확인 → left_click(coordinate)
방법 2: Tab/Shift+Tab 반복으로 순차 이동
```
- **⚠ Ctrl+F로 셀 진입 금지**: 표 내부 텍스트를 못 찾는 HWP 버그 있어 신뢰 불가
- screenshot 기반 클릭 시: 셀 텍스트 중앙을 정확히 클릭 (테두리 근처 불가)

## C. 셀 내용 입력

1. 원하는 셀로 이동 (Tab 또는 클릭)
2. 텍스트 입력:
   ```
   computer(action="type", text="셀 내용", tabId={탭ID})
   ```
3. 다음 셀로 이동: Tab

**여러 셀에 순차 입력 패턴:**
```
type("A1 내용") → Tab
type("B1 내용") → Tab
type("C1 내용") → Tab  ← 다음 줄 첫 셀로 자동 이동
type("A2 내용") → Tab
... 반복
```

## D. 줄/칸 추가/삭제

### 단축키 (표 내부에서) ★ 도움말 확인
```
줄 추가:     Ctrl+Enter     ← 현재 행 아래에 새 행 추가
줄 지우기:   Ctrl+Backspace ← 커서 위치 행 전체 삭제
칸 추가:     Alt+Insert     ← 현재 열 오른쪽에 새 열 추가 (웹에서 확인 필요)
칸 지우기:   Alt+Delete     ← 커서 위치 열 전체 삭제 (웹에서 확인 필요)
```
- **⚠ 마지막 1행만 남은 상태**: Ctrl+Backspace 불가 → Shift+Esc로 표 탈출 후 표 개체 선택 → Delete
- **Tab (마지막 셀에서)**: 자동으로 새 행 추가

### 메뉴 경로
1. 표 안에 커서 위치
2. 표 메뉴 → "줄/칸 추가하기" 또는 "줄/칸 지우기"
3. 추가/삭제 위치/수 설정 → 확인

### 표 삭제 (전체)
```
방법 1 (빠름): 표를 클릭하여 개체로 선택 (8개 핸들) → Delete
방법 2 (행 반복): Ctrl+Backspace 반복 → 마지막 행은 Shift+Esc → Backspace
```

## E. 셀 블록 선택 ★ CDP 드래그로 가능 (검증 완료)

셀 블록 선택은 CDP `left_click_drag`로 자동화 가능. **시작점이 셀 경계선 교차점(corner) 근처**여야 한다.

**⚠ 핵심 원리**: 셀 내부 텍스트에서 드래그 → **텍스트 선택**. 셀 경계선 교차점에서 드래그 → **셀 블록 선택**.

### 셀 블록 선택 알고리즘
```
1. 표가 화면에 보이도록 스크롤 → screenshot 촬영

2. screenshot에서 **내부** 셀 경계선 위치 파악:
   [세로 경계선 x] 각 칼럼 사이의 세로 테두리선 x좌표
   [가로 경계선 y] 각 행 사이의 가로 테두리선 y좌표
   ★ 모든 내부 경계선 교차점에서 작동 (헤더|첫행 외에도 가능)
   ⚠ 표 외곽(좌상단/우하단) 경계선은 불가 → 텍스트 선택됨

3. 드래그 시작점 결정 (screenshot 좌표):
   - x: 아무 내부 세로 경계선 x (±3px 이내)
   - y: 아무 내부 가로 경계선 y (±3px 이내)
   ★ 시작점 = 내부 경계선 교차점! 텍스트 중앙·외곽 경계 금지!
   ⚠ 경계선에서 시작 시 양쪽 셀 모두 블록에 포함됨

4. 끝점: 블록 범위 마지막 행 중앙 y, 마지막 칼럼 중앙 x

5. left_click_drag(start_coordinate=[시작], coordinate=[끝])

6. 상태바 확인: "0/N" = 셀 블록 성공 | 다른 값 = 실패

7. 실패 시: 시작점 x,y를 ±3px 조정, 최대 3회 재시도
   → 3회 실패 시 사용자에게 수동 셀 블록 안내
```

### 셀 블록 선택 후 사용 가능한 기능 (모두 검증 완료)
| 단축키 | 기능 | 검증 |
|--------|------|------|
| `Ctrl+←/→` | 블록 내 칼럼 너비 조정 | ✅ |
| `Ctrl+↑/↓` | 블록 내 행 높이 조정 | ✅ |
| `W` | 블록 내 셀 너비를 같게 | ✅ |
| `H` | 블록 내 셀 높이를 같게 | ✅ |
| `M` | 블록 내 셀 합치기 | ✅ |
| `S` | 셀 나누기 다이얼로그 | ✅ |
| `P` | 표/셀 속성 다이얼로그 | ✅ |
| 블록 계산식 | 합계/평균/곱 | ✅ |

**⚠ 핵심 구분: Ctrl 없는 단축키(P, S, M, W, H)는 셀 블록 상태에서만 작동**
- 셀 편집 모드(커서가 셀 안에서 텍스트 편집 중): P→"p" 텍스트 입력, S→"s" 텍스트 입력
- 셀 블록 모드(상태바 "0/N"): P→표/셀 속성, S→셀 나누기, M→합치기, W→너비균등, H→높이균등
- **⚠ W/H는 블록 범위 내 셀에만 적용** (표 전체가 아님)
- **⚠ 좌표 하드코딩 금지**: 표 위치는 문서/줌/스크롤마다 다름 → 반드시 screenshot 기반
- **⚠ 좌표 정밀도**: ±3px 이내 필수 (2px 차이로 실패/성공 갈린 사례 확인)

### F5 키는 여전히 불가
- **F5**: 브라우저 새로고침으로 가로채임 (JS/CDP 우회 모두 실패 확정)
- **JS synthetic event**: `isTrusted=false` → WASM 무시
- **마우스 드래그(CDP)**: 위 알고리즘으로 대체

### 셀 블록 실패 시 대안 (중요도순)
1. **좌표 미세 조정 후 재시도** (±5px, 최대 3회)
2. **처음부터 올바른 구조로 설계** → 합치기 불필요하게
3. **사용자에게 수동 조작 안내**: "마우스로 셀 드래그하여 블록 선택 후 M/W/H 키"
4. **표 삭제 후 재생성**: 구조 변경이 클 때

## F. 셀 나누기 (S) ★ 자동화 가능

단일 셀 나누기는 셀 블록 선택 없이 **커서 위치만으로 실행 가능** → 완전 자동화 가능.

### 단축키
```
표 안에서: 표 메뉴 열기 → S (셀 나누기)
```

### 자동화 절차
```
1. 나눌 셀에 커서 이동 (Tab/Shift+Tab 또는 Ctrl+F)
2. 표 메뉴 → 셀 나누기:
   find(query="표 메뉴") → left_click
   find(query="셀 나누기") → left_click
3. 셀 나누기 다이얼로그:
   - 줄 개수: 분할할 행 수 (기본 2)
   - 칸 개수: 분할할 열 수 (기본 2)
   - □ 줄 높이를 같게 나누기
   - □ 셀을 합친 후 나누기
4. 나누기 버튼 클릭
```

### 주의사항
- 최대 63개까지 분할 가능
- 셀 블록 선택 시 다중 셀 동시 나누기도 가능하나 → 웹 에디터에서는 단일 셀만

## G. 셀 크기 조정 ★ 핵심 기능

### 방법 1: 표/셀 속성 다이얼로그 (정밀)
```
1. 크기 조정할 셀에 커서 위치
2. 표 메뉴 → 표/셀 속성 (또는 셀 블록 상태에서 P키)
   ⚠ P키는 셀 블록 모드에서만 작동! 셀 편집 모드에서는 "p" 텍스트 입력됨
3. "셀" 탭 선택
4. "셀 크기 적용" 체크 → 너비(mm)/높이(mm) 입력
5. 확인
```
- **⚠ 중요**: 셀 크기 = 해당 **줄 전체 높이** + **칸 전체 너비** 적용 (개별 셀 불가)
- **⚠ 제한**: 현재 값보다 작게 설정 불가 (내용에 의해 최소 크기 결정됨)

### 방법 2: 글자 크기에 맞춘 행 높이 자동 조정
```
표 만들기 시 높이="자동" 설정 → 셀 내 글자 크기에 따라 행 높이 자동 확장
→ 대부분의 경우 별도 높이 조정 불필요
```
- 글자 크기를 변경하면 행 높이가 자동으로 맞춰짐
- 수동 고정 높이가 필요한 경우에만 셀 속성에서 직접 지정

### 방법 3: 표 전체 안 여백 일괄 설정
```
표/셀 속성 → "표" 탭 → "모든 셀의 안 여백" → 좌/우/위/아래 일괄 입력
```
- 표 내 글자와 셀 경계 사이 간격을 통일

### 셀 높이/너비 균등 (셀 블록 필요 → 섹션 E 알고리즘 사용)
```
셀 높이를 같게: H (셀 블록 상태에서)
셀 너비를 같게: W (셀 블록 상태에서)
Ctrl+←/→: 칼럼 너비 미세 조정 (셀 블록 상태에서)
Ctrl+↑/↓: 행 높이 미세 조정 (셀 블록 상태에서)
```
- ⚠ W/H는 블록 범위 내 셀에만 적용 (표 전체가 아님)
- 셀 블록 선택: 섹션 E의 CDP 드래그 알고리즘 참조
- 셀 블록 실패 시 대안: 표/셀 속성에서 각 행/열의 크기를 수치로 동일하게 입력

## H. 표/셀 속성

### 표 속성 다이얼로그 열기
1. 표 안에 커서 위치
2. 표 메뉴 → 표/셀 속성:
   ```
   find(query="표 메뉴", tabId={탭ID}) → left_click
   find(query="표/셀 속성", tabId={탭ID}) → left_click
   ```
   또는: 표 선택 후 `find(query="개체 속성")` → 클릭

### 속성 탭 구성 (6개 탭)

**1. 기본 탭:**
- 크기: 너비(mm), 높이(mm), □크기고정
- 위치: □글자처럼취급, 본문과의배치(어울림/자리차지/글앞으로/글뒤로)
- 본문위치: 왼쪽/오른쪽/양쪽/큰쪽
- 가로: 기준(단), 정렬(왼쪽), 기준값(mm)
- 세로: 기준(문단), 정렬(위쪽), 기준값(mm)
- ☑쪽영역안으로제한, □서로겹침허용, □개체와 조판부호를 항상 같은 쪽에

**2. 여백/캡션 탭:**
- 바깥 여백: 왼쪽/오른쪽/위쪽/아래쪽 (표 가장자리~본문 간격)
- 캡션: 위치(위/아래/왼쪽/오른쪽), 개체와의 간격(기본 3mm), □여백까지 확대, □한줄로입력

**3. 테두리 탭:** (표 전체 선택 시)
- 표 외곽/내부 테두리선 스타일, 굵기, 색상

**4. 배경 탭:** (표 전체 선택 시)
- 표 배경색, 배경 그림

**5. 표 탭:** ★ 여러 쪽 표에 중요
- 쪽 경계에서: 나눔 / 셀 단위로 나눔 / 나누지 않음
- □제목 줄 자동 반복 (셀 탭에서 "제목 셀" 설정 필수)
- 자동 나뉜 표의 경계선: 선 종류/굵기/색
- 모든 셀의 안 여백: 왼/오/위/아래 일괄 설정

**6. 셀 탭:** ★ 셀 크기/속성의 핵심
- □셀크기적용: 너비(mm), 높이(mm)
  - ⚠ 해당 셀이 속한 **줄 전체의 높이/칸 전체의 너비**에 동일 적용
  - ⚠ 현재 값보다 작게 설정 불가
- □안여백지정: 왼쪽/오른쪽/위쪽/아래쪽 (mm) → 표 전체 여백 무시하고 이 값 적용
- 속성: □셀보호 (커서 진입 차단), □제목셀 (여러 쪽 표에서 자동 반복), 세로정렬(위/가운데/아래)
- 필드: 필드이름, □양식모드에서편집가능

### ⚠ 셀 크기 조정 방법
- **표/셀 속성 다이얼로그 사용 (권장)**: 셀 탭 → 셀 크기 적용 체크 → 너비/높이 입력
- **마우스 드래그로 열 경계 리사이즈**: 웹 에디터에서는 텍스트 선택과 혼동되기 쉬움, 비추천
- 표/셀 속성 진입: 표 안에 커서 → 표 메뉴 → 표/셀 속성

## I. 테두리/배경 설정

### 셀 테두리 변경
1. 셀 또는 셀 블록 선택 (Shift+방향키 또는 마우스 드래그)
2. 표 메뉴 → 테두리/배경:
   ```
   find(query="테두리/배경", tabId={탭ID}) → left_click
   ```
3. 테두리 다이얼로그:
   - 선 스타일 선택 (실선, 점선, 이중선 등)
   - 선 굵기 설정
   - 선 색상 설정
   - 적용 범위: 바깥, 안쪽, 각각 등
4. 확인 클릭

### 셀 배경 변경
1. 셀 선택
2. 테두리/배경 다이얼로그 → 배경 탭
3. 배경색 선택 또는 그라데이션/그림 설정
4. 확인

## J. 블록 계산식

### 셀 블록 필요 → 섹션 E 알고리즘으로 블록 선택 후 사용
블록 계산식(합계, 평균, 곱)과 1,000단위 구분쉼표는 **셀 블록 선택이 전제 조건**:
1. 섹션 E의 CDP 드래그로 계산 대상 셀 범위를 블록 선택
2. 표 메뉴 → 블록 계산식 → 합계/평균/곱 선택

### 셀 블록 실패 시 대안
- 합계/평균 등은 **수동으로 계산 후 직접 입력**
- 사용자에게 수동 작업 안내: 마우스 드래그로 셀 블록 선택 → 표 메뉴 → 블록 계산식

## 에러 처리

### 표 메뉴가 비활성
- 커서가 표 안에 있어야 표 관련 메뉴가 활성화됨
- 표 안 아무 셀 클릭 후 재시도

### 셀 블록 선택이 안 됨
- **⚠ F5 사용 불가** (브라우저 새로고침). Shift+방향키 또는 마우스 드래그로 블록 선택
- screenshot으로 선택 상태 시각적 확인

### 셀 합치기 후 레이아웃 깨짐
- 되돌리기(Ctrl+Z)로 원복 후 다른 방법 시도
- 표 전체를 새로 만드는 것도 고려

## K. 배치 셀 입력 (★ 속도 최적화)

표 데이터를 빠르게 입력하기 위한 최적화 패턴.

### 행 단위 배치 입력 (권장)
```
[패턴] 한 행의 셀을 연속 type+Tab으로 빠르게 입력

행1: type("셀1") → key(Tab) → type("셀2") → key(Tab) → type("셀3") → key(Tab)
행2: type("셀4") → key(Tab) → type("셀5") → key(Tab) → type("셀6") → key(Tab)
...반복

★ wait 최소화: 한글 입력 후 Tab 전에만 wait(0.1), 영문/숫자는 wait 불필요
★ 마지막 셀에서 Tab → 자동으로 다음 행 첫 셀로 이동
```

### 헤더+데이터 분리 입력 패턴
```
[1단계: 헤더 행]
type("항목") → Tab → type("2023년") → Tab → type("2024년") → Tab → type("증감률") → Tab

[2단계: 데이터 행 반복]
type("매출") → Tab → type("1,200") → Tab → type("1,500") → Tab → type("25%") → Tab
type("비용") → Tab → type("800") → Tab → type("900") → Tab → type("12.5%") → Tab
```

### 대용량 표 입력 시 주의
- 20행 이상: 10행마다 screenshot으로 입력 상태 확인
- 숫자 데이터만 있는 행: wait 없이 연속 입력 가능
- 한글 포함 행: 각 셀 입력 후 Tab 전 wait(0.1)

## L. 헤더 행 서식 워크플로우 (★ 검증 완료)

**⚠ 볼드 서식은 Tab 이동 시 다음 셀로 전파되지 않음** → 각 셀마다 개별 적용 필수.

### 방법 A: 각 셀에서 서식+입력 동시 (권장)
```
[패턴] Ctrl+B → type → Tab → Ctrl+B → type → Tab ... 반복

1. A1: Ctrl+B → type("항목") → wait(0.1) → Tab
2. B1: Ctrl+B → type("2023년") → wait(0.1) → Tab
3. C1: Ctrl+B → type("2024년") → wait(0.1) → Tab
4. D1: Ctrl+B → type("증감률") → wait(0.1) → Tab → 다음 행으로

★ 가운데 정렬이 필요하면 각 셀에서 Ctrl+B 전에 정렬 먼저 변경
★ 제목 셀 설정 (여러 쪽 표): 표/셀 속성 → 셀 탭 → "제목 셀" 체크
```

### 방법 B: 전체 입력 후 서식 소급 적용
```
[1단계: 텍스트 먼저 전부 입력]
type("헤더1") → Tab → type("헤더2") → Tab → type("헤더3") → Tab

[2단계: 첫 헤더 셀로 복귀]
Shift+Tab 반복 (헤더 칼럼 수만큼) 또는 첫 셀 직접 클릭

[3단계: 각 셀에서 서식 적용]
Ctrl+A (셀 전체 선택) → Ctrl+B → Tab → 반복
```

### 배경색 적용
```
셀 블록 선택 불가 → 셀별 개별 배경 설정 필요
→ 방법: 표 메뉴 → 테두리/배경 → 배경 탭 → 색상 선택 → 확인 (셀마다 반복)
→ 실용적 대안: 볼드+가운데 정렬만으로 헤더 구분 (배경색 생략)
→ 배경색이 반드시 필요하면 사용자에게 수동 설정 안내
```

## M. 캡션 + 출처 통합 패턴

보고서 표에 캡션과 출처를 함께 작성하는 표준 패턴.

### 표 캡션 전체 워크플로우
```
[표 위: 캡션]
1. 표 삽입 위치에서 표 제목 스타일 적용 (Ctrl+숫자)
2. type("<표 {장번호}-{순번}> {제목}") → Enter
3. (단위 필요시) type("(단위: 억 원, %)") → 오른쪽 정렬 → Enter

[표 생성 + 데이터 입력]
4. 표 만들기 (Alt+N → T → 행/열 수 → 확인)
5. 배치 셀 입력 (섹션 K 패턴)

[표 아래: 주석/출처]
6. Shift+Esc로 표 탈출 (또는 표 아래 클릭) → 본문 커서 복귀
7. 출처 스타일 적용 (Ctrl+숫자 또는 작은 글씨 직접 설정)
8. (주석 있으면) type("주: {설명}") → Enter
9. type("자료: {기관명}({연도}), {자료명}") → Enter
10. Enter (빈 줄) → 본문 스타일 복귀 → 다음 내용 계속
```

### 그림 캡션 패턴
```
[그림 삽입 후]
1. 그림 아래로 커서 이동
2. 그림 제목 스타일 적용
3. type("[그림 {장번호}-{순번}] {제목}") → Enter
4. 출처 스타일 적용
5. type("자료: {출처}") → Enter
```

### 캡션 번호 체계
```
<표 N-M>    N=장번호, M=해당 장 내 순번 (예: <표 3-1>, <표 3-2>)
[그림 N-M]  동일 규칙 (예: [그림 5-1], [그림 5-2])
장이 없는 문서: <표 1>, <표 2> ... 순번만 사용
```

## N. 칼럼 너비 지정

### 방법 A: 표/셀 속성 다이얼로그 (정밀)
```
1. 너비 조정할 칼럼의 아무 셀 클릭
2. 표 메뉴 → 표/셀 속성 → 셀 탭
3. "셀 크기 적용" 체크 → 너비(mm) 입력
4. 확인
5. 다른 칼럼도 동일 반복
```

### 방법 B: 균등 분배 사용 (간편)
```
대부분의 보고서 표는 기본 균등 분배로 충분
특정 칼럼만 넓게: 해당 칼럼 셀에서 표/셀 속성 → 너비 지정
나머지 칼럼은 자동 균등 배분
```

### 일반적인 칼럼 너비 패턴 (A4 기준)
```
항목명 칼럼: 30~40mm (좌측 고정)
데이터 칼럼: 20~30mm (균등)
비고/합계 칼럼: 25~35mm
전체 표 너비: 용지 너비 - 좌우 여백 (A4: ~150mm)
```
