## 주요 단축키 요약

| 단축키 | 기능 |
|--------|------|
| `Ctrl+Z` | 되돌리기 |
| `Ctrl+Shift+Z` | 다시 실행 |
| `Ctrl+X` | 오려두기 |
| `Ctrl+C` | 복사 |
| `Ctrl+V` | 붙이기 |
| `Ctrl+Alt+C` | 모양 복사 |
| `Ctrl+E` | 지우기 |
| `Ctrl+A` | 모두 선택 |
| `Ctrl+F10` | 문자표 |
| `Ctrl+K+E` | 필드 입력 |
| `Ctrl+K+H` | 하이퍼링크 |
| `Ctrl+K+B` | 책갈피 |
| `Alt+L` | 글자 모양 |
| `Alt+T` | 문단 모양 |
| `Alt+K+N` | 문단 번호 모양 |
| `Alt+N+I` | 그림 삽입 |
| `Alt+N+B` | 글상자 |
| `Alt+N+T` | 표 만들기 |
| `Alt+N+L` | 문단띠 |
| `Alt+N+S` | 현재 쪽만 감추기 |
| `F5` | ⚠ 셀 블록 선택 (웹에서는 브라우저 새로고침으로 작동! 사용 불가) |
| `F6` | 스타일 |
| `F7` | 편집 용지 |
| `Ctrl+Enter` | 쪽 나누기 |
| `Ctrl+Shift+Enter` | 단 나누기 |
| `Ctrl+Alt+Enter` | 다단 설정 나누기 |
| `Ctrl+Num-` | 한 수준 증가 |
| `Ctrl+Num+` | 한 수준 감소 |
| `Tab` | 표 내 다음 셀 이동 |
| `Shift+Tab` | 표 내 이전 셀 이동 |
